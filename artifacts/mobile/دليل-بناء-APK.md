# دليل تحويل التطبيق إلى ملف APK

هذا الملف يشرح لك خطوة بخطوة كيف تبني ملف APK لتطبيق "PS4 Hall Manager" وتثبته على جهاز أندرويد، وذلك باستخدام خدمة EAS Build المجانية من Expo (تعمل من جهازك، خارج Replit).

## المتطلبات
- جهاز كمبيوتر عليه Node.js مثبت (يفضل نسخة 20 أو أحدث).
- حساب مجاني على expo.dev (سجّل من https://expo.dev/signup إذا لم يكن لديك حساب).
- اتصال بالإنترنت (البناء يتم على سيرفرات Expo، لا حاجة لتثبيت Android Studio).

## الخطوات

### 1. فك ضغط المشروع
فك ضغط الملف الذي حصلت عليه (mobile-app.zip) في أي مجلد على جهازك.

### 2. افتح Terminal داخل مجلد المشروع
```bash
cd مسار-المجلد-الذي-فككت-فيه-الضغط
```

### 3. ثبّت الحزم (Dependencies)
```bash
npm install
```

### 4. ثبّت أداة EAS CLI
```bash
npm install -g eas-cli
```

### 5. سجّل الدخول بحساب Expo
```bash
eas login
```
أدخل بريدك الإلكتروني وكلمة المرور الخاصة بحساب expo.dev.

### 6. اربط المشروع بحسابك (أول مرة فقط)
```bash
eas init
```
سيسألك عن اسم المشروع، اضغط Enter لقبول الاسم الافتراضي.

### 7. ابنِ ملف APK
```bash
eas build --platform android --profile preview
```
- سيتم رفع الكود لسيرفرات Expo وبناء الـ APK هناك (يستغرق عادة 10-15 دقيقة).
- بعد الانتهاء، سيعطيك رابط مباشر لتحميل ملف الـ APK.

### 8. تثبيت التطبيق على الهاتف
- افتح الرابط الذي حصلت عليه من هاتف الأندرويد مباشرة، أو حمّل الملف على الكمبيوتر ثم انقله للهاتف.
- فعّل خيار "السماح بالتثبيت من مصادر غير معروفة" في إعدادات الهاتف إذا طُلب منك ذلك.
- افتح ملف APK وثبّته.

## ملاحظات مهمة
- هذه الخدمة (EAS Build) مجانية لعدد محدود من عمليات البناء شهريًا، وهذا كافٍ لتطبيق شخصي أو تجريبي.
- التطبيق يخزّن كل بياناته محليًا على الجهاز (AsyncStorage) ولا يحتاج إلى إنترنت أو سيرفر خارجي بعد التثبيت.
- إذا واجهت أي خطأ أثناء البناء، انسخ رسالة الخطأ وأرسلها لي هنا وسأساعدك في حلها.
- في حال احتجت لاحقًا بناء نسخة لمتجر Google Play، استخدم نفس الأمر لكن مع profile الاسمه `production` بدلاً من `preview`.
